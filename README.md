# Python String Methods

A beginner-friendly Python project demonstrating commonly used **string methods** with simple examples.

This repository helps students understand how Python string methods work using basic functions and print statements.

---

## Topics Covered

### Case Conversion Methods

* `lower()`
* `upper()`
* `title()`
* `capitalize()`
* `swapcase()`

### Checking Methods

* `islower()`
* `isupper()`
* `istitle()`
* `isalpha()`
* `isalnum()`
* `isdigit()`
* `isspace()`

### Searching Methods

* `startswith()`
* `endswith()`
* `count()`
* `find()`

### Space Handling Methods

* `strip()`
* `lstrip()`
* `rstrip()`

### String Manipulation Methods

* `split()`
* `join()`
* `replace()`

---

## Example

```python
def low(string):
    return string.lower()

print(low("HELLO WORLD"))
```

Output:

```bash
hello world
```
